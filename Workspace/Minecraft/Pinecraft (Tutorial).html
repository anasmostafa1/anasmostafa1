Pinecraft game tutorial:

  • W: Walk forward

  • S: Walk back

  • A: Walk left

  • D: Walk right

  • Space: Jump

  • Left mouse key: Place block

  • Right mouse key: Destroy block

  • T: Teleport to random position

  • F: Flying mode

  • H: Exit flying mode

  • G: Change block

  • Escape: Exit game

  • F11: Full screen mode
